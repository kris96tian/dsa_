WEB SEQ

A web app created using C++ code by calling it with the Python Framework Flask. Bioinformatics agorithms and files are used to analyse biological data such as DNA, RNA, and protein sequences. The toolset includes sequence alignment, pattern matching with Aho-Corasick, Horspool, or even with Q-Gram indices. A section of BLAST for generating neighborhood and a simple structure prediction of RNA molecules are also showcased.
